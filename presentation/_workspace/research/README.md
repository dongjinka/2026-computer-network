# _input — 팀 조사 자료를 여기에 넣으세요

하네스가 "기존 조사 기반 개선"을 하려면 팀이 노션에 정리한 내용이 필요합니다.

## 노션 → Markdown export 방법

1. 노션에서 해당 페이지(또는 상위 페이지) 우상단 `···` 메뉴 클릭
2. **Export** 선택
3. Format: **Markdown & CSV**, Include subpages: 체크(하위 페이지 포함 시)
4. 다운로드된 `.zip`을 풀어서, 그 안의 `.md`(와 이미지 폴더)를 **이 `_input/` 폴더에 그대로 복사**

```
_input/
├── DTN-조사.md
├── DTN-조사/           (이미지 등 첨부 폴더, 있으면)
└── (기타 하위 페이지 .md)
```

## (선택) 다른 팀 QA 질문 리스트

다른 팀의 발표 후 Q&A 질문 리스트가 있으면 `_input/other-team-qa.md`로 넣어주세요.
qa-defense-strategist가 그 질문 패턴·난이도를 분석해, 우리 발표에 나올 법한 까다로운 질문을
미리 시뮬레이션하고 슬라이드를 단단하게 다듬습니다. (없으면 자체 7개 카테고리로 생성)

## 배치 후

이 폴더에 export를 넣은 뒤, Claude에게 **"DTN 발표 자료 만들어줘"** 라고 요청하면
오케스트레이터가 이 자료를 감사·보강·차별화해 Marp 슬라이드(`presentation.md`)와
Q&A 준비 시트(`_workspace/06_qa_prep.md`)를 제작합니다.

> 이미지가 많아 export가 번거로우면, 노션 텍스트를 채팅에 붙여넣어도 됩니다.
> 그래도 진행하기 어려우면 "그냥 새로 조사해줘"라고 하면 신규 조사 모드로 진행합니다.
